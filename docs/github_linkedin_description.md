# GitHub / LinkedIn Project Description

## GitHub Short Description

TalentPulse is an end-to-end Talent Acquisition Analytics Command Center built with Excel, SQL, Power BI concepts, and Python automation to analyze hiring funnels, source quality, recruiter productivity, offer conversion, time-to-hire, and skill demand using a realistic synthetic recruitment dataset.

## LinkedIn Post Draft

Excited to share **TalentPulse: AI-Assisted Talent Acquisition Analytics Dashboard** — a project I built to combine analytics, SQL, Excel, Power BI dashboard design, and Python automation in a realistic hiring intelligence use case.

The project analyzes:
- Hiring funnel conversion and stage-wise drop-off
- Source/channel effectiveness
- Recruiter productivity
- Offer acceptance and time-to-hire
- Skills and talent intelligence
- Weekly TA reporting automation

Tech used: Excel, SQL, Power BI concepts, Python, SQLite, DAX, CSV automation, dashboarding, and recruitment analytics.

This helped me practice how raw recruitment data can be transformed into decision-ready insights for smarter and faster hiring.
