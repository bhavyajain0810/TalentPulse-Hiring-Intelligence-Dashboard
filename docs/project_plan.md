# Project Plan: TalentPulse Hiring Intelligence Command Center

## 1. Project Title

**TalentPulse: AI-Assisted Talent Acquisition Analytics Dashboard**

## 2. Summary

TalentPulse is a complete recruitment analytics project that turns raw hiring data into decision-ready insights. It combines Excel reporting, SQL analytics, Power BI dashboard design, and Python automation to help a TA team monitor pipeline health, source quality, recruiter productivity, hiring velocity, and talent intelligence.

## 3. Real-World Problem Statement

Recruitment teams manage candidates across multiple channels, roles, recruiters, and interview stages. Without a centralized analytics model, it becomes difficult to understand which sources produce quality candidates, where candidates drop off, how fast roles are being filled, and which skills are available in the pipeline.

TalentPulse solves this by creating a realistic analytics layer for:
- Hiring funnel optimization
- Candidate sourcing and channel effectiveness
- Offer conversion and decline analysis
- Recruiter productivity tracking
- Skills and talent intelligence
- Diversity/location/department insights
- Weekly automated reporting

## 4. Stakeholders

- Talent Acquisition Managers
- Recruiters
- Campus Hiring Teams
- HR Analytics Teams
- Business Hiring Managers
- Internship/early-career program teams

## 5. Core Questions Answered

1. How many candidates are entering each stage of the funnel?
2. Which stage has the highest drop-off?
3. Which source has the best quality score?
4. Which recruiter is handling the strongest pipeline?
5. Which jobs/departments are slowest to close?
6. Which skills appear most frequently among shortlisted or hired candidates?
7. What is the offer acceptance rate?
8. How can weekly reporting be automated?

## 6. Deliverables

- Normalized SQL database
- Raw and processed CSV datasets
- Excel analytics workbook
- Power BI dashboard blueprint
- DAX measure library
- Python automation scripts
- Documentation and interview talking points
